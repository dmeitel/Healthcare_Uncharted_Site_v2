This page ships with a read-only token baked in, so normally you never see this card. If you're reading it, that token got revoked or GoatCounter rejected it. Mint a replacement, then update EMBEDDED_TOKEN in the page source so it sticks for every browser, or paste it below for a this-browser-only fix (lives in localStorage).

  1. Open GoatCounter → API tokens
  2. New token. Check only "Read statistics".
  3. Paste it below.
