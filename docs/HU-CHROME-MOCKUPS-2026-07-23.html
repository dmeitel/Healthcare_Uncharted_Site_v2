The call

Build A. Steal B's always-grouped sections for the drawer's Display tab, and steal C's discipline as the target state: if a control can become a map label instead of chrome, it becomes a map label. A is also the only one that keeps one grammar across desktop and phone, which is what made the maps feel right once navigation was fixed.
